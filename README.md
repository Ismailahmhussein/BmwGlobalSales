# Power BI Dashboard - BMW Global Sales

## Overview
This project is an interactive Power BI dashboard analyzing BMW global sales data from 2018–2025.

## Key Features
- Sales trends over time  
- Regional performance comparison  
- Key performance indicators (KPIer) such as revenue, units sold, and average price  
- Interactive filtering by model, year, and region  

## Tools & Technologies
- Power BI  
- Data modeling and transformation
